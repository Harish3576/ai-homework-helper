# AI Homework Helper (Groq) — No Login, No Payments

✅ Next.js 14 + Tailwind + Prisma (SQLite)
✅ Groq LLM homework solver (Hindi + English)
✅ Support form (no login)
✅ Simple daily limit per device/IP (in-memory for dev)

## Run (Windows CMD)
```bat
npm install
copy .env.example .env.local
notepad .env.local
npm run db:push
npm run dev
```

Open: http://localhost:3000"# ai-homework-helper"
